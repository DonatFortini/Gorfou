Random forest

Une Random forest est un méta estimateur qui adapte un certain nombre de classificateurs d'arbre de décision sur divers sous-échantillons de l'ensemble de données et utilise la moyenne pour améliorer la précision prédictive et contrôler le surajustement. La taille du sous-échantillon est contrôlée avec le paramètre max_samples si bootstrap=True (par défaut), sinon l'ensemble de données entier est utilisé pour construire chaque arbre.
